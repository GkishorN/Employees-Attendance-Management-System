ADMIN HELP

ADMIN LOGIN:By entering Admin Id and Password Admin Login can be done.

ADMIN REGISTRATION:Admin can add new admin also.

REMOVE DETAILS:Details of Admin can be removed through this button.

ADD EMPLOYEE:Admin can add new employee to the system.

VIEW EMPLOYEE DETAIL: By clicking this button details of employee can be viewed.And also Editing,Deleting process also provided here.

ATTENDENCE DETAILS:The total presence,total absence and total worked hours of employees will be viewed by clicking this button.

EMPLOYEE TIME REGISTER:By clicking this button admin can view the time of attendence marked by employee,Break time,Leaving time,lunch time.

CHECK PRESENCE:Admin can check the presence of Employee by entering the date and his Office ID.

RESET SYSTEM:Admin should press this button either before starting of the day or after end of the day.
